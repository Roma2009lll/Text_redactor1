# Text Editor with UNDO/REDO

A simple text editor implemented in C with full UNDO/REDO functionality.

## Features
- ✅ Add text
- ✅ Delete last N characters  
- ✅ UNDO/REDO operations
- ✅ Save/Load to file
- ✅ History tracking

## Usage
Compile and run the program. Use the menu to perform text operations.
